はてな教科書 JavaScript 編
================================================================

## 講義の目的

* JavaScript について、自分で調べて学習するための基礎知識を身につける
  * リファレンスをひけば良い部分は覚えない


## 話の流れ

* JavaScript の言語コア部分
  * 構文とか
* Web と JS にまつわるあれこれ
  * DOM
  * 非同期プログラミング
  * Ajax
  * jQuery
  * フロントエンド設計
  * Node.js


駆け足で進めるので、速すぎるなら遠慮せず言ってください！
