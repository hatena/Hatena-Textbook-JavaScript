言語的特徴

  • 動的型付け
  • 関数も値の一種
  • C-like な文法

  • プロトタイプベースの継承構造

    • 近年 Class 構文が追加された

  • 言語のコア部分は ECMAScript として標準化されている

    • JavaScript と ECMAScript 仕様について (JavaScript Overview - MDN)
    • ECMAScript 最新仕様は ECMAScript® 2016 Language Specification
