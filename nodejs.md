## Node.js とは

- ブラウザの外で動く JavaScript 処理系
  - Chrome 用に開発された V8 という実行エンジンを使っている
- 使い道
  - CLI ツール
  - Web サーバー / アプリケーションサーバー
  - 開発用ツール
- はてなでも多くのプロジェクトで使われています
  - Node.js の使い方を覚えておいて損はない
